# TodoAppKT

A modern to-do app built with Jetpack Compose, Material 3, Room, StateFlow, and Hilt.

## Highlights
- Large top app bar with live task counts and subtle transitions
- Elevated task cards with gradient accent stripe and completed-state styling
- Edit tasks in-place with updated timestamps
- Search and filter chips with active/done counts
- Polished add-task bottom sheet with validation and keyboard actions
- Custom Material 3 theme (teal + slate palette, rounded shapes, refined typography)
- Edge-to-edge layout and lifecycle-aware StateFlow collection

## Tech stack
- Kotlin
- Jetpack Compose (Material 3)
- Room
- Hilt
- StateFlow

## Getting started
1. Install Android Studio (latest stable) and JDK 17.
2. Open the project in Android Studio and let Gradle sync.
3. Run the `app` configuration on an emulator or device.
